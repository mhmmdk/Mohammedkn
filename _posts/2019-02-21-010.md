<h3>Problem 10</h3>
<p>The sum of the primes below 10 is 2 + 3 + 5 + 7 = 17.
<br><br>Find the sum of all the primes below two million.</p>
</div>

<h3> Solution 1</h3>

Let's look at the answer:

```python
n = 3
primes = []
total = 2

# no multiples of 2
while n < 2000000:
    isPrime = True
    sqrt_n = n**0.5

    for p in primes:

        if p > sqrt_n:
            break

        if not n%p:
            isPrime = False
            break

    if isPrime:
        primes += [n]
        total += n

    n += 2


print(total)
```
